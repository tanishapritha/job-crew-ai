# Job Engine Frontend (React)

Professional frontend for the Job Automation Engine.

## Features
- **React + Vite**: High-performance development.
- **Lucide Icons**: Crisp, professional iconography.
- **Dark Mode**: Integrated light/dark toggle.
- **Live Search**: Integrated "Search Now" button for instant job discoveries.
- **Responsive**: Grounded design that works on mobile and desktop.

## Local Development
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Deployment (Vercel)

### Option 1: Vercel CLI
1. `npm i -g vercel`
2. `vercel` (inside the `frontend` directory)

### Option 2: GitHub Dashboard
1. Connect your repo.
2. Set "Root Directory" to `frontend`.
3. Vercel will auto-detect everything else.
